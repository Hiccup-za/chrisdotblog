### Added
- Favicon support with custom web favicon
- Next.js app icon configuration

## [0.1.0] - 2026-02-08

### Added
- Turborepo monorepo structure with workspace configuration
- Next.js web application in `apps/web`
- Hero, Projects, and SocialLinks components
- Theme context and configuration system
- Tailwind CSS styling setup
- TypeScript configuration for monorepo
- Turbo.json with build, dev, lint, typecheck, and test tasks
- Comprehensive README with setup and usage instructions
- Package manager configuration (Bun)

### Changed
- Updated README with Turborepo monorepo documentation and structure
